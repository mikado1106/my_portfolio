# Personal Portfolio

Personal portfolio website built with Next.js to showcase projects, skills, and professional background.

## Live

**https://my-portfolio-six-zeta-78.vercel.app**

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js |
| Language | JavaScript |
| Deployment | Vercel |

## Getting Started

```bash
# Clone
git clone https://github.com/mikado1106/my_portfolio.git
cd my_portfolio

# Install dependencies
npm install

# Run
npm run dev
```
